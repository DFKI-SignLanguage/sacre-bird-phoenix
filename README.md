# sacre-bird-phoenix
Companion repository for annotations, back translations, benchmark analyses, and supporting materials from our audit of RWTH-PHOENIX-2014T as a benchmark for German Sign Language translation.

## Overview
This repository accompanies our work on corpus and benchmark quality in RWTH-PHOENIX-2014T.

It is intended to share:
- annotations from the corpus audit
- back translations and related metadata

## Important note
This repository is intended to share derived audit resources and does not redistribute the original RWTH-PHOENIX-2014T dataset itself.
